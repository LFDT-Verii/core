# @verii/validation

Validation package for Velocity Projects, based on ajv and fastify validation

## Installation

```bash
yarn add @verii/validation
```

## Usage

```js
const pkg = require('@verii/validation');

// Use the exported members for your workflow.
```

## Entry Point

- `index.js`

## Notes

- Unable to enumerate exports automatically in a standalone runtime: `Cannot find module 'ajv/dist/2019'`
- Refer to `index.js` for the package API.

## Development

Run from the repository root:

```bash
yarn workspace @verii/validation test
yarn workspace @verii/validation lint
```
