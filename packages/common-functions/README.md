# @verii/common-functions

Set of functions and utils used by Verii projects

## Installation

```bash
yarn add @verii/common-functions
```

## Usage

```js
const pkg = require('@verii/common-functions');

// Use the exported members for your workflow.
```

## Entry Point

- `index.js`

## Notes

- Unable to enumerate exports automatically in a standalone runtime: `Cannot find module 'lodash/fp'`
- Refer to `index.js` for the package API.

## Development

Run from the repository root:

```bash
yarn workspace @verii/common-functions test
yarn workspace @verii/common-functions lint
```
