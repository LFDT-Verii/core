# @verii/endpoints-event-processing

Event processing

## Installation

```bash
yarn add @verii/endpoints-event-processing
```

## Usage

```js
const pkg = require('@verii/endpoints-event-processing');

// Use the exported members for your workflow.
```

## Entry Point

- `src/index.js`

## Notes

- Unable to enumerate exports automatically in a standalone runtime: `Cannot find module '@fastify/autoload'`
- Refer to `src/index.js` for the package API.

## Development

Run from the repository root:

```bash
yarn workspace @verii/endpoints-event-processing test
yarn workspace @verii/endpoints-event-processing lint
```
