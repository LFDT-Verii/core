# @verii/tests-helpers

Generic helpers for tests

## Installation

```bash
yarn add @verii/tests-helpers
```

## Usage

```js
const pkg = require('@verii/tests-helpers');

// Use the exported members for your workflow.
```

## Entry Point

- `index.js`

## Notes

- Unable to enumerate exports automatically in a standalone runtime: `Cannot find module 'expect'`
- Refer to `index.js` for the package API.

## Development

Run from the repository root:

```bash
yarn workspace @verii/tests-helpers test
yarn workspace @verii/tests-helpers lint
```
