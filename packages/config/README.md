# @verii/config

Generic configuration file for Velocity Projects

## Installation

```bash
yarn add @verii/config
```

## Usage

```js
const pkg = require('@verii/config');

// Use the exported members for your workflow.
```

## Entry Point

- `index.js`

## Notes

- Unable to enumerate exports automatically in a standalone runtime: `Cannot find module 'env-var'`
- Refer to `index.js` for the package API.

## Development

Run from the repository root:

```bash
yarn workspace @verii/config test
yarn workspace @verii/config lint
```
