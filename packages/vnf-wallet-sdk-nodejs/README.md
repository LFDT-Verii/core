# @verii/vnf-nodejs-wallet-sdk

VNF Wallet SDK Nodejs

## Installation

```bash
yarn add @verii/vnf-nodejs-wallet-sdk
```

## Usage

```js
const pkg = require('@verii/vnf-nodejs-wallet-sdk');

// Use the exported members for your workflow.
```

## Entry Point

- `dist/index.js`

## Notes

- Unable to enumerate exports automatically in a standalone runtime: `Cannot find module '/tmp/verii-issue-278-readmes/packages/vnf-wallet-sdk-nodejs/dist/index.js'`
- Refer to `dist/index.js` for the package API.

## Development

Run from the repository root:

```bash
yarn workspace @verii/vnf-nodejs-wallet-sdk test
yarn workspace @verii/vnf-nodejs-wallet-sdk lint
yarn workspace @verii/vnf-nodejs-wallet-sdk build
```
