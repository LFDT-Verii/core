# @verii/image-processing

Set of image processing functions used in Verii projects

## Installation

```bash
yarn add @verii/image-processing
```

## Usage

```js
const { ExtensionTypes, MimeToExtensionType, getImageFormat } = require('@verii/image-processing');

// Use the exported members for your workflow.
```

## Entry Point

- `index.js`

## Top-level Exports

- `ExtensionTypes`
- `MimeToExtensionType`
- `getImageFormat`
- `getMimeType`

## Development

Run from the repository root:

```bash
yarn workspace @verii/image-processing test
yarn workspace @verii/image-processing lint
```
