# @verii/endpoints-credential-types-registrar

Credential Types Registrar

## Installation

```bash
yarn add @verii/endpoints-credential-types-registrar
```

## Usage

```js
const pkg = require('@verii/endpoints-credential-types-registrar');

// Use the exported members for your workflow.
```

## Entry Point

- `src/index.js`

## Notes

- Unable to enumerate exports automatically in a standalone runtime: `Cannot find module '@fastify/autoload'`
- Refer to `src/index.js` for the package API.

## Development

Run from the repository root:

```bash
yarn workspace @verii/endpoints-credential-types-registrar test
yarn workspace @verii/endpoints-credential-types-registrar lint
```
