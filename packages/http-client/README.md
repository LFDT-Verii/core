# @verii/http-client

HTTP client for Verii network

## Installation

```bash
yarn add @verii/http-client
```

## Usage

```js
const { initCache, initHttpClient } = require('@verii/http-client');

// Use the exported members for your workflow.
```

## Entry Point

- `index.js`

## Top-level Exports

- `initCache`
- `initHttpClient`

## Development

Run from the repository root:

```bash
yarn workspace @verii/http-client test
yarn workspace @verii/http-client lint
```
