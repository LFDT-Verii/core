# @verii/common-schemas

Set of validation schemas

## Installation

```bash
yarn add @verii/common-schemas
```

## Usage

```js
const { addressSchema, didServiceSchema, errorSchema } = require('@verii/common-schemas');

// Use the exported members for your workflow.
```
