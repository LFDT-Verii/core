# @verii/db-kms

KMS plugin that uses a db

## Installation

```bash
yarn add @verii/db-kms
```

## Usage

```js
const { dbKmsPlugin, defaultRepoOptions, initDbKms } = require('@verii/db-kms');

// Use the exported members for your workflow.
```

## Entry Point

- `index.js`

## Top-level Exports

- `dbKmsPlugin`
- `defaultRepoOptions`
- `initDbKms`
- `kmsRepo`

## Development

Run from the repository root:

```bash
yarn workspace @verii/db-kms test
yarn workspace @verii/db-kms lint
```
