# @verii/logger

Logger for Verii network

## Installation

```bash
yarn add @verii/logger
```

## Usage

```js
const { loggerProvider } = require('@verii/logger');

// Use the exported members for your workflow.
```

## Entry Point

- `index.js`

## Top-level Exports

- `loggerProvider`

## Development

Run from the repository root:

```bash
yarn workspace @verii/logger test
yarn workspace @verii/logger lint
```
