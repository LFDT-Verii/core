# @verii/blockchain-functions

Blockchain related wrappers and helpers.

## Installation

```bash
yarn add @verii/blockchain-functions
```

## Usage

```js
const pkg = require('@verii/blockchain-functions');

// Use the exported members for your workflow.
```

## Entry Point

- `index.js`

## Notes

- Unable to enumerate exports automatically in a standalone runtime: `Cannot find module 'lodash/fp'`
- Refer to `index.js` for the package API.

## Development

Run from the repository root:

```bash
yarn workspace @verii/blockchain-functions test
yarn workspace @verii/blockchain-functions lint
```
