# @verii/organizations-registry

Velocity DID documents library

## Installation

```bash
yarn add @verii/organizations-registry
```

## Usage

```js
const pkg = require('@verii/organizations-registry');

// Use the exported members for your workflow.
```

## Entry Point

- `index.js`

## Notes

- Unable to enumerate exports automatically in a standalone runtime: `Cannot find module 'lodash/fp'`
- Refer to `index.js` for the package API.

## Development

Run from the repository root:

```bash
yarn workspace @verii/organizations-registry test
yarn workspace @verii/organizations-registry lint
```
