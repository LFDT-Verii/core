# @verii/vc-checks

Velocity Verifiable credentials checks functions

## Installation

```bash
yarn add @verii/vc-checks
```

## Usage

```js
const pkg = require('@verii/vc-checks');

// Use the exported members for your workflow.
```

## Entry Point

- `index.js`

## Notes

- Unable to enumerate exports automatically in a standalone runtime: `Cannot find module 'date-fns/fp'`
- Refer to `index.js` for the package API.

## Development

Run from the repository root:

```bash
yarn workspace @verii/vc-checks test
yarn workspace @verii/vc-checks lint
```
